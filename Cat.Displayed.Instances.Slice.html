The canonical self-indexingüîó

There is a canonical way of viewing any category B\ca{B}B as displayed over itself, given fibrewise by taking slice categories. As a fibrationüîó

If (and only if) B\ca{B}B has all pullbacks, then the self-indexing B\ca{B}B is a Cartesian fibration.
